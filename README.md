# 🛒 ShopEasy

A full-stack e-commerce web application built using React.js and Spring Boot.

## Features

- User Authentication
- Product Management
- Admin Dashboard
- Shopping Cart
- Order Management
- REST APIs

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript

### Backend
- Spring Boot
- Java
- MySQL

## Installation

### Frontend

```bash
npm install
npm start
```

### Backend

```bash
mvn spring-boot:run
```
